# Multi-Language Translator Web App

## Overview

A simple web-based language translator that allows users to translate text between multiple languages using JavaScript and a translation API.

## Features

* Translate text between multiple languages
* User-friendly interface
* Copy translated text
* Text-to-speech support
* Multiple language selection

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

```
index.html
style.css
script.js
countries.js
copy.svg
speaker.svg
```

## How to Run

1. Download or clone the repository.
2. Open `index.html` in a web browser.
3. Enter text and select languages.
4. Click the **Translate** button.
